# arena.ai Floating Code Copy & Minimize Enhancer

A Tampermonkey / Greasemonkey userscript that enhances the user experience on [lmarena.ai](https://lmarena.ai) and [arena.ai](https://arena.ai) with several useful features.

## Features

- **Floating Copy Button for Code Blocks**: A floating "Copy Code" button appears next to visible code blocks for easy copying.
- **Minimize/Maximize Buttons**:
  - Individual minimize buttons for AI model messages.
  - Individual minimize buttons for user messages.
  - Global buttons to minimize/maximize all model chats or all user messages.
- **Chat Management**:
  - Copy all loaded chats to clipboard.
  - Download the entire chat as a `chat.txt` file (sorted by position).
- **Tooltips for Truncated Labels**: Hover over truncated text labels (like model names) to see the full text in a tooltip.
- **Multi-language Support**: Supports English, Arabic, and Persian (Farsi) with proper RTL direction.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) extension for your browser.
2. Click on the following link to install directly:
   - [Install Script](https://raw.githubusercontent.com/mohammad021/arena-ai-floating-code-copy-minimize-enhancer/main/script.js)
3. Or copy the script content and create a new script in Tampermonkey.

The script will automatically update when a new version is released (thanks to `@updateURL`).

## Usage

- After installation, visit https://lmarena.ai or https://arena.ai.
- Code blocks will show a floating copy button.
- Minimize buttons appear on messages.
- Global controls are added to the sidebar footer.
- Hover on truncated labels for full text.
